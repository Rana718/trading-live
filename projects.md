【具体的な要件】

① リアルタイムチャート表示
・TradingView または MT5 のチャート画面をリアルタイムで配信に載せる
・主要通貨ペア（USD/JPY、EUR/USDなど）のチャートを自動で切り替え表示
・テクニカル指標（移動平均線、ボリンジャーバンド等）の表示

② AI音声によるチャート解説ナレーション
・VOICEVOX等のAI合成音声を使用
・チャートの値動き・トレンドを自動分析し、リアルタイムで音声解説を生成
・例：「現在ドル円は〇〇円付近で推移、上昇トレンドが継続中です」

③ 経済ニュースの自動読み上げ
・経済ニュースサイトから最新ニュースを自動取得（スクレイピング）
・取得したニュースをAI音声で読み上げ
・チャート解説の合間に定期的にニュースを挟む構成

④ テロップ・字幕の自動表示
・読み上げ内容をリアルタイムで字幕表示
・通貨ペア名・現在価格・変動率などの情報テロップ
・ニュース見出しのテロップ表示

⑤ YouTubeライブへの自動配信
・OBS Studio等を使ったYouTubeへのリアルタイム自動配信
・配信が途切れた場合の自動復旧機能
・24時間安定稼働する仕組み

⑥ VPS上での完全自動稼働
・Windows VPS（新規でも可）上で全てが自動で動く環境構築
・リモートデスクトップでこちらが操作する必要なし
・PC起動時の自動スタート、エラー時の自動復旧



【技術スタック（想定・提案歓迎）】

・VPS: Windows VPS（Contabo等／新規構築可）
・チャート取得: TradingView / MT5
・ニュース取得: 経済ニュースサイトからのスクレイピング
・音声合成: VOICEVOX / その他AI TTS
・映像合成: OBS Studio / FFmpeg 等
・配信: YouTube Live API / RTMP
・自動化: Python / Node.js / バッチスクリプト等
